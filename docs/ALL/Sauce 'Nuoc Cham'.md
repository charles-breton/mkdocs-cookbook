# INGREDIENTS

- [ ] 2 gousses d’ail émincées
- [ ] 1 c. thé piment rouge broyé
- [ ] 3 c. tab. sucre
- [ ] 2 c. tab. jus de lime
- [ ] 4 c. tab. sauce au poisson
- [ ] 1 t. eau

---
# INSTRUCTIONS

1. Dans un bol de grosseur moyenne mélanger tous les ingrédients. Ajuster les quantités au goût.
